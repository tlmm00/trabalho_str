## Especificação do trabalho
- Desenvolver um simulador de eventos discretos que gera o escalonamento de um conjunto de tarefas periódicas.
- A implementação deverá proporcionar pelo menos dois algoritmos de escalonamento diferentes e deverá incrementar o tempo pelos intervalos separando os eventos.
- A saída será um diagrama de Gantt, podendo ser em caráteres Ascii ou interface gráfica.
